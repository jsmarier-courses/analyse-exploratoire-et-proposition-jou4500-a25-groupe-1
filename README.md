### Analyse exploratoire de données (AED) et proposition

Vous trouverez ici les fichiers pour le projet de session des cours JOU4500 Journalisme numérique II à l'Université d'Ottawa.

Dans cette analyse exploratoire, nous étudions les heures de départ au travail à Ottawa à partir du recensement détaillé de 2021. Les données, importées dans Google Sheets, couvrent les lignes 2570 à 2595, incluant les heures de départ, les moyens de transport, la durée des trajets et les destinations selon les quartiers. Nous nettoyons d’abord les données pour éliminer doublons et erreurs. L’objectif est d’identifier les heures de pointe et les facteurs qui les influencent.
